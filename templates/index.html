Inputs

Plans per NIC controls NIC breakout: 0 = single plan (all NICs in one fabric), 1 = 1x400G, 2 = 2x200G, 4 = 4x100G (for 400G NICs). For values above 0, total plans = plans_per_nic × NICs per GPU.

Breakout cables are assumed when a port runs faster than what it connects to. A super-spine tier is only introduced when the 2-tier spine-leaf can't fan out wide enough for the requested GPU count.
